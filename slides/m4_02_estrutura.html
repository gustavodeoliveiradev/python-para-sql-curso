4.1 Estruturando um Projeto de Automação

Antes de automatizar qualquer coisa, vale separar o script em partes previsíveis — assim como você separaria uma query grande em CTEs.

Pasta/ArquivoPapel
dados/Arquivos de entrada (CSV, Excel, .db)
saida/Relatórios e exports gerados pelo script
script.pyO código que faz o trabalho
.envSenhas e configurações sensíveis (nunca vai pro Git!)
requirements.txtLista de bibliotecas que o script precisa
requirements.txt
pandas
sqlite3  # já vem com o Python, não precisa listar
python-dotenv
schedule
💡 Com um requirements.txt, qualquer pessoa (ou qualquer servidor) instala tudo que o script precisa com um único comando: pip install -r requirements.txt.
