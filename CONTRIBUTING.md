# 🤝 Como Contribuir

Obrigado pelo interesse em contribuir! Este curso é feito com carinho e toda ajuda é bem-vinda.

## 📋 Tipos de Contribuição

- 🐛 **Reportar bugs** nos slides ou exercícios
- ✨ **Sugerir melhorias** no conteúdo ou design
- 📝 **Adicionar exercícios** extras
- 🌍 **Traduções** para outros idiomas

## 🚀 Passo a Passo

1. **Fork** este repositório
2. Crie uma **branch** para sua feature:
   ```bash
   git checkout -b feature/minha-melhoria
   ```
3. Faça suas alterações e **commit**:
   ```bash
   git commit -m "✨ Adicionei exercício sobre list comprehension"
   ```
4. **Push** para sua branch:
   ```bash
   git push origin feature/minha-melhoria
   ```
5. Abra um **Pull Request**

## ✅ Checklist antes de enviar

- [ ] Testei os slides no navegador
- [ ] Verifiquei se o CSS está funcionando
- [ ] Revisiei ortografia e gramática
- [ ] Mantive o estilo visual consistente
