## 🛠️ Como Usar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/gustavodeoliveiradev/python-para-sql-curso.git
```

2. Abra o `index.html` no navegador:
```bash
cd python-para-sql-curso
# No Windows:
start index.html
# No Mac:
open index.html
# No Linux:
xdg-open index.html
```

3. Ou use o Live Server do VS Code (extensão recomendada!)
